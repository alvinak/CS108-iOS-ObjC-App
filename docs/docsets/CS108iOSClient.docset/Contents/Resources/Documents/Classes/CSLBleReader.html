CSLBleReader

@interface CSLBleReader : CSLBleInterface

This object class holds the core function of the reader API and it allows developers to configure and control the device being connnected. It is a sub-class of CSLBleInterface, which handles the low-level Bluetooth LE connectiviites.

    - (BOOL)setInventoryConfigurations:(QUERYALGORITHM)inventoryAlgorithm
                      MatchRepeats:(Byte)match_rep
                         tagSelect:(Byte)tag_sel
                  disableInventory:(Byte)disable_inventory
                           tagRead:(Byte)tag_read
                      crcErrorRead:(Byte)crc_err_read
                            QTMode:(Byte)QT_mode

                     inventoryMode:(Byte)inv_mode;

    Swift

    func setInventoryConfigurations(_ inventoryAlgorithm: QUERYALGORITHM, matchRepeats match_rep: UInt8, tagSelect tag_sel: UInt8, disableInventory disable_inventory: UInt8, tagRead tag_read: UInt8, crcErrorRead crc_err_read: UInt8, qtMode QT_mode: UInt8, tagDelay tag_delay: UInt8, inventoryMode inv_mode: UInt8) -> Bool

    Parameters

    inventoryAlgorithm

    Inventory algorithm to use.
    match_rep

    Stop after N tags inventoried (zero indicates no stop)
    tag_sel

    1 = enable tag select prior to inventory, read, write, lock or kill. 0 = no select issued.
    disable_inventory

    Do not run inventory
    tag_read

    0 = no tag read issued 1 = enable read 1 bank after inventory 2 = enable read 2 banks after inventory
    crc_err_read

    0 = disable crc error read 1 = enable crc error read
    QT_mode

    0 = disable QT temporary read private EPC 1 = enable QT temporary read private EPC
    tag_delay

    Time delay for each tag in ms (use to reduce tag rate). Value should be between 1 to 65
    inv_mode

    0 = normal mode 1 = compact mode

    Return Value

    TRUE if the operation is successful
